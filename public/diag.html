Live read-out of /models/manoel-island.glb and its sidecar. Use this to figure out whether tiling is in the data, in the GLB, or in the renderer.

1 · Heightmap from sidecar JSON

Pixel = one heightmap cell. North is up. Greyscale = elevation (black = sea-floor, white = highest). If you see the LiDAR pattern repeated side-by-side here, it's a data bug (build script). If this looks like one clean island but the 3D scene is tiled, it's a renderer bug.

Colours rescaled per render. Blue overlay = cells outside the Manoel outline (forced to sea-floor).
